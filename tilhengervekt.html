ℹ️ Regler for Tilhengervekt i Norge

Førerkort klasse B (vanlig bilførerkort):

  • Totalvekt bil + tilhenger kan IKKE overstige 3500 kg
  • Tilhengerens totalvekt kan IKKE overstige bilens egenvekt
  • Tilhengerens vekt må IKKE overstige bilens tilhengerkapasitet

Førerkort klasse BE:

  • Totalvekt bil + tilhenger kan overstige 3500 kg
  • Tilhengervekt kan være inntil 3500 kg
  • Tilhengerens totalvekt kan overstige bilens egenvekt

Viktig å huske:

  • Overskridelse av tillatt vekt er ulovlig
  • Forsikringen kan nekte å dekke skader ved ulovlig vekt
  • Sjekk alltid vognkortet for nøyaktige tall
